Summary: Latitude vs. X

This project is to analyze what the weather is like as you get closer to or farther away from the equator. To show this, we pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

Matplotlib is then used to plot latitude vs. temperature, cloudiness, wind speed, and humidity. This site provides the dataset/visualizations and explains any correlations.
